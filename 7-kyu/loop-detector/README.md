<h2><a href=https://www.codewars.com/kata/68851563123e161332d2a84b/train/csharp target="_blank">Loop Detector</a></h2><h3>7 kyu</h3><h1 id="loop-detector">Loop Detector</h1><p>In this kata, you'll simulate traversing a one-way pointer chain, similar to a singly linked list. Each element in the input list represents the index of the next element to move to.</p><p>Your task is to determine whether this chain eventually enters a loop, or whether it terminates by stepping out of bounds.</p><h2 id="function-signature">Function Signature</h2><ul><li><p><code>arr</code> is a list or array of non-negative integers.</p></li><li><p>Return <code>True</code> (or your language's equivalent) if the traversal enters a loop.</p></li><li><p>Return <code>False</code> (or your language's equivalent) if the traversal terminates.</p></li></ul><h2 id="how-traversal-works">How Traversal Works</h2><ul><li><p>Begin at index <code>0</code>.</p></li><li><p>Read the value at the current index to get the next index.</p></li><li><p>Continue stepping through the array.</p></li><li><p>If you encounter an index you've already visited, you've found a loop — return <code>True</code> (or your language's equivalent).</p></li><li><p>If a step takes you to an index outside the array, the traversal ends — return <code>False</code> (or your language's equivalent).</p></li></ul><h2 id="examples">Examples</h2><pre><code>Input: 1, 2, 3, 4, 2 ➞ TruePath: 0 -&gt; 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 2 -&gt; ... (loop detected)Input: 1, 2, 3, 4, 5 ➞ FalsePath: 0 -&gt; 1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 5 -&gt; (out of bounds)Input: 0 ➞ TruePath: 0 -&gt; 0 -&gt; 0 -&gt; ... (self-loop)Input: 3, 2, 1, 6 ➞ FalsePath: 0 -&gt; 3 -&gt; 6 -&gt; (out of bounds)Input: 1, 0 ➞ TruePath: 0 -&gt; 1 -&gt; 0 -&gt; 1 -&gt; ... (cycle of two)</code></pre><h2 id="notes">Notes</h2><ul><li><p>The input may be empty. In that case, return <code>False</code> (or your language's equivalent).</p></li><li><p>All elements are guaranteed to be non-negative integers.</p></li><li><p>You must detect any kind of loop, whether it be a:</p><ul><li><p>Self-loop (e.g., <code>[0]</code>)</p></li><li><p>Two-node loop (e.g., <code>[1, 0]</code>)</p></li><li><p>Larger cycle (e.g., <code>[1, 2, 0]</code>)</p></li></ul></li><li><p>An index is considered out of bounds if it is greater than or equal to <code>len(arr)</code>.</p></li></ul>
